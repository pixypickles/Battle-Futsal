# Battle Futsal Prototype v0.5

リングゴール向けに、シュートとパスの高さを補助するよう変更した試作版です。

## v0.5の変更点
- Bを離した時にシュート。短く押してもリングへ向かう放物線になる
- B長押しでチャージ。最大付近ではリングの高さへ正確に合う
- チャージ中は選手の上にゲージが表示され、狙うリングが発光する
- シュートの難しさを「高さ調整」よりも、方向・位置取り・妨害へ移した
- A短押しは低いパス、A長押しは敵を越えやすいロブパス
- 空中のルーズボール付近でBを押すと、ダイレクトキックを狙える

## 操作
- 左スティック / WASD・矢印: 移動・向き
- ボールなし: A(J)=剣、B(K)=盾体当たり（空中ボール付近ではダイレクトキック）、C(L)=ガード
- ボールあり: A(J)=パス（長押しでロブ）、B(K)=シュート（長押しでチャージ）、C(L)=ボールガード

`index.html` をブラウザで開くと動きます。GitHub Pagesにもそのまま配置できます。
