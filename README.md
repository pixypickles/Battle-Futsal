# Battle Futsal Prototype v0.2

斜め上から見下ろす疑似3D表示へ変更した試作版です。

## 変更点
- 円形コートを奥行きのある楕円として表示
- コート外周を低い壁として表現
- ゴールを地面から浮いた縦向きリングに変更
- ボールに高さ・重力・バウンドを追加
- シュートは山なりに飛び、リング内部を通過した場合のみ得点

## 操作
- 左スティック / WASD・矢印: 移動
- ボールなし: A(J)=剣、B(K)=盾体当たり、C(L)=ガード
- ボールあり: A(J)=低いパス、B(K)=リングへのシュート、C(L)=ボールガード

`index.html` をブラウザで開くと動きます。GitHub Pagesにもそのまま配置できます。
