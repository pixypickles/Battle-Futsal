# Battle Futsal Prototype v0.6

リングゴール向けのシュート補助と、空中の高さが見て分かるボール表現を追加した試作版です。

## v0.6の変更点
- ボールが高く上がるほど画面上で大きく表示される
- 高度が上がるほど影が小さく薄くなり、地面との距離が分かりやすい
- ボールの物理的な当たり判定は変えず、見た目だけを拡大
- Bのチャージ量に応じて、蹴った後のボールがリング方向へ徐々にカーブする
- 短押しはほぼ向いている方向へ飛び、長押しほど方向補正が強くなる
- 補正は完全な自動ゴールではなく、大きく外した方向からは入らない
- シュートの高さはリングへ届く放物線を自動計算

## 操作
- 左スティック / WASD・矢印: 移動・向き
- ボールなし: A(J)=剣、B(K)=盾体当たり（空中ボール付近ではダイレクトキック）、C(L)=ガード
- ボールあり: A(J)=パス（長押しでロブ）、B(K)=シュート（長押しで方向補正）、C(L)=ボールガード

`index.html` をブラウザで開くと動きます。GitHub Pagesにもそのまま配置できます。
