# Battle Futsal Prototype v0.7

2対2化と、空中シュートが見えなくなる問題を修正した試作版です。

## v0.7の変更点
- 青チームに味方AI、赤チームに敵AIを1人ずつ追加し、2対2へ変更
- 味方AIはボール保持中のプレイヤーへ近づく敵を押し返し、シュートの隙を作る
- 味方AIへのパスと、AI同士のパス・シュートを追加
- シュートのチャージ完了時間を約0.5秒へ短縮
- チャージ中の移動速度は少し低下するが、完全停止はしない
- 攻撃を受けるとシュートチャージが解除される
- シュートの高さ計算を安定化し、異常に高く飛んで画面から消えないよう修正
- 空中のボールに短い軌跡を追加し、移動方向を追いやすくした
- ボール高度に安全上限を設け、数値的な暴走でも描画範囲から消えないようにした

## 操作
- 左スティック / WASD・矢印: 移動・向き
- ボールなし: A(J)=剣、B(K)=盾体当たり（空中ボール付近ではダイレクトキック）、C(L)=ガード
- ボールあり: A(J)=パス（長押しでロブ）、B(K)=シュート（長押しで方向補正）、C(L)=ボールガード

`index.html` をブラウザで開くと動きます。GitHub Pagesにもそのまま配置できます。
