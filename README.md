# Combat Airball Prototype 2.6 — Classic Monk

## 修正
- キャラクターを初期プロトタイプ寄りのシンプルなイラスト調へ変更
- 頭・胴・手足を太い輪郭線で描き、スマホでも動作を読みやすく調整
- 全員モンクで、長棍を常に両手で握っている表示
- 横突きと真下への振り下ろしで棍の向きが大きく変化
- 横吹っ飛び後は壁で完全に停止
- 毎フレーム座標をアリーナ内へ固定し、画面外へ消えない安全処理を追加

## 操作
- A / D または ← / →：移動
- J：横突き
- S / ↓ ＋ J：空中振り下ろし
- K：キック / 空中ボレー
- L または Space：ジャンプ
- R：リセット
