# Combat Airball v0.1

軽装3対3の空中スポーツ・プロトタイプです。

## ルール
- 各チームは忍者、棍モンク、手甲モンクの3人。
- 各陣地に高いリングが2つ。
- Bでボールをトス。
- トスだけではリングに届かず、空中キックだけが得点判定を持つ。
- トスした本人は、そのトスから得点できない。
- 味方のトスだけでなく敵のトスでも得点可能。
- 壁際でジャンプすると壁ジャンプ。忍者は特に高く飛ぶ。

## 操作
- 移動: WASD / 矢印 / 仮想スティック
- A / J: クラス固有攻撃
- B / K: トス
- C / L: ジャンプ。空中でボールの近くならジャンプキック

## クラス
- Ninja: クナイ投げ。高速で空中妨害。
- Staff Monk: 足払い。地上の広い範囲を制圧。
- Gauntlet Monk: パンチ。空中でも相手を弾く。
