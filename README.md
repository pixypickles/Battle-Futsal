# Battle Futsal Prototype v1.0

センターサークルからの簡単な大量得点を抑え、4クラスを試せる版です。

## v1.0の変更点
- ゴール可能になる「シュートゾーン」を追加。黄色いラインより相手側へ進むか、十分近い位置から撃ったシュートだけ得点になる
- 味方AIもセンター付近から即シュートせず、シュートゾーンまで運ぶ・パスするよう調整
- 画面上部のクラスボタンでプレイヤーを切り替え可能
- Knight: 木刀の長いスタン、盾体当たり、ガード
- Mace: 遅いが非常に強いノックバック。AもBも重量級
- Monk: 長い間合いのスタッフ攻撃。Cはステップ移動
- Ninja: 高速移動、素早い短剣、Bで中距離の手裏剣、Cでダッシュ
- 味方AIはMace、敵AIはNinjaとMonkになり、クラス差を試合中に確認できる

## 操作
- 左スティック / WASD・矢印: 移動・向き
- ボールなし: A(J)=主攻撃、B(K)=副攻撃、C(L)=ガードまたは移動技
- ボールあり: A(J)=パス、B(K)=シュート、C(L)=ボールガード
- 上部のクラス名: Knight → Mace → Monk → Ninja の順に切り替え

`index.html` をブラウザで開いてください。
