# Battle Futsal Prototype v0.1

武器で相手に隙を作り、キックで得点する近未来の武装球技プロトタイプです。

## 今回入っているもの

- 1対1（プレイヤー：ナイト / 相手：簡易AI）
- 壁あり楕円コート
- センサー型ゴール
- 仮想アナログスティック
- A / B / C の状態切り替え操作
- ルーズボールを剣や盾で弾く処理
- 攻撃を受けても負傷せず、短時間よろけるだけ
- 2分間のスコア戦

## 操作

### ボールなし

- A：剣を振る
- B：盾で体当たり
- C：ガード（移動可能だが、ボール所持時は低速キープ）

### ボールあり

- A：前方へパス
- B：相手ゴールへシュート
- C：シールドキープ

### PCでの確認

- 移動：WASD または矢印
- A：J
- B：K
- C：L

## ローカルで起動

`index.html` をブラウザで開くだけでも動きます。ローカルサーバーを使う場合：

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

## GitHub Pagesで公開

1. GitHubで新しいリポジトリを作成
2. このフォルダの中身をアップロード
3. Repository Settings → Pages
4. Sourceを `Deploy from a branch` にする
5. Branchを `main`、フォルダを `/ (root)` にして保存

数分後、表示されたURLからスマホでも遊べます。

## 設計上のメモ

今回は処理を軽くするため、画像素材や物理エンジンを使わず、HTML5 Canvasの基本図形だけで描画しています。キャラクターも線と単純な形だけで表現し、アニメーション枚数を増やさずに、ノックバックとヒット反応で手応えを出す方針です。
