# Battle Futsal Prototype v0.3

操作欄と攻撃・ゴール判定を調整した試作版です。

## v0.3の変更点
- スマホの画面下でA/B/Cボタンが切れないよう、`100dvh`・セーフエリア・操作欄の高さを調整
- 剣と盾体当たりのノックバックを大幅に強化
- ヒットストップ、画面振動、放射状エフェクト、文字エフェクトを追加
- リングを小さくし、高さを上げた
- シュートの完全自動照準を廃止し、進行方向を基本にした弱い照準補助へ変更
- AIシュートにわずかな狙いの誤差を追加
- リング中央をきれいに通過した場合だけ得点

## 操作
- 左スティック / WASD・矢印: 移動・向き
- ボールなし: A(J)=剣、B(K)=盾体当たり、C(L)=ガード
- ボールあり: A(J)=パス、B(K)=シュート、C(L)=ボールガード

`index.html` をブラウザで開くと動きます。GitHub Pagesにもそのまま配置できます。
