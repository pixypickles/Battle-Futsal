# Battle Futsal Prototype v1.2

ボールの視認性と、キャラクター移動のメリハリを改善したゲームフィール調整版です。

## v1.2の変更点
- ボールを約1.24倍に拡大し、太い二重縁取りと淡い発光を追加
- ルーズボールの真下に黄色い着地点リングと高さガイドを常時表示
- ボール影を大きく濃くし、空中でも位置を追いやすく調整
- プレイヤーとAIの移動を即時等速から加速・急減速方式へ変更
- 反転時は一瞬速度が落ち、向きが滑らかに切り替わるよう調整
- 歩幅、上下動、左右の足運びを追加し、滑走感を軽減
- 動き出しに小さな踏み込みを追加

## 引き継いだv1.1の変更点
- 移動速度を再調整：Ninja > Monk > Knight > Mace
- ノックバック耐性を追加：MaceとKnightは重く、Ninjaは大きく飛ばされる
- Knight A：木刀なぎ払い（長スタン） / B：盾タックル（大ノックバック）
- Mace A：広い横なぎ / B：振り下ろし（超ノックバック）
- Mace Bで壁へ叩きつけると追加スタン
- Maceの武器描画を修正し、球状の頭が先端に来るよう変更
- Monk A：低い足払い / B：長射程の突き
- Ninja B：大型で回転する手裏剣を実体弾として表示し、白い軌跡を追加
- 遠距離シュートほどリング方向への自動補正が弱くなるよう調整

## 操作
- 左スティック / WASD・矢印：移動・向き
- ボールなし：A(J)=主攻撃、B(K)=強攻撃、C(L)=ガードまたは移動技
- ボールあり：A(J)=パス、B(K)=シュート、C(L)=ボールガード
- 上部のクラス名：Knight → Mace → Monk → Ninja の順に切り替え

`index.html` をブラウザで開いてください。
