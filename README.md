# Combat Airball Prototype 2.3 — Air Combat

空中戦の密度を上げた調整版です。

## 変更点
- ボールへのキック判定を大幅に拡大
- ボールの上下位置によって、キック後の軌道が大きく変化
- 空中ではキックと武器を押し続けて繰り返し使用可能
- キックにも対人当たり判定を追加
- 空中キック命中時に軽い反動を得て、連続攻撃しやすく変更
- 選手同士が空中でぶつかると弾かれる
- 棍の叩き落とし判定を拡大
- クナイの当たり判定を拡大し、短いひるみへ変更
- 命中エフェクトを追加

## 操作
- A / D または ← / →: 移動
- J: 武器
- K: キック / 空中ボレー
- L または Space: ジャンプ / 空中壁ジャンプ
- R: リセット
