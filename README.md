# Combat Airball Prototype 2.4 — Sword & Lift

## 変更点
- 棍キャラを片手剣ファイターへ変更
- 剣は斜め下への振り下ろしで相手を落とす
- 剣の軌跡エフェクトを追加
- 地上キックの上方向速度を大幅に強化
- ボール重力をさらに弱くし、確実に空中戦へ移行
- キック判定をさらに拡大
- 丸みのあるマスコット風の頭身へ調整
- 頭と胴を大きくし、スマホでも見分けやすく変更

## 操作
- A / D または ← / →: 移動
- J: 武器（クナイ / 片手剣）
- K: キック / 空中ボレー
- L または Space: ジャンプ / 空中壁ジャンプ
- R: リセット
