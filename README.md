# Battle Futsal Prototype v0.4

ルーズボールの回収と壁際の挙動を調整した試作版です。

## v0.4の変更点
- 壁に当たったボールの横方向の速度を減衰し、外周に沿って長時間回り続けないように変更
- 壁の連続接触時はさらに強く減速
- 地面を転がるボールの停止判定を追加
- 低速のルーズボールに対する回収範囲を拡大
- プレイヤーがボールへ向かっている場合は、少し速いボールも拾いやすく調整
- 高速ボールに接触した場合は、すり抜けずにトラップして大きく減速

## 操作
- 左スティック / WASD・矢印: 移動・向き
- ボールなし: A(J)=剣、B(K)=盾体当たり、C(L)=ガード
- ボールあり: A(J)=パス、B(K)=シュート、C(L)=ボールガード

`index.html` をブラウザで開くと動きます。GitHub Pagesにもそのまま配置できます。
