# Battle Futsal Prototype v0.9

味方AIの判断と位置取りを重点的に改善した2対2の試作版です。

## v0.9の変更点
- 味方AIがボール保持者の後ろを追うのではなく、前方・斜め前へ回ってパスコースを作る
- プレイヤーがボールを持つと、近い敵を優先して押し返しシュートレーンを空ける
- ルーズボールでは、最寄りの味方が確実に回収へ向かう
- ボールの速度を予測して少し先の位置へ走り、壁際のボールにも追いつきやすくした
- 守備時は敵を後ろから追うのではなく、自陣ゴール側へ回り込んでシュートコースを塞ぐ
- ゴール前ではボール方向を向いてガードする緊急守備を追加
- 味方AIが持ったときは、長く立ち止まらず短いチャージで素早くシュートする
- 囲まれたときは、より空いている味方、特にプレイヤーへのパスを優先
- AIのドリブルが敵へ直進し続けないよう、相手を避ける斜め移動を追加

## 操作
- 左スティック / WASD・矢印: 移動・向き
- ボールなし: A(J)=剣、B(K)=盾体当たり（空中ボール付近ではダイレクトキック）、C(L)=ガード
- ボールあり: A(J)=パス（長押しでロブ）、B(K)=シュート（長押しで方向補正）、C(L)=ボールガード

`index.html` をブラウザで開くと動きます。GitHub Pagesにもそのまま配置できます。
